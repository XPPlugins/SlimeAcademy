本插件致力于在`BukkitAPI`框架下实现 **《魔法禁书目录》** 及其衍生作品中出现的超能力<p>
基于`Slimefun`以提供电力系统，与道具获取系统<p>
若不安装`Slimefun`，您需要自行实现玩家能力的开发过程、道具的获取。
